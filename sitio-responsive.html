Haciendo un sitio responsive

Se utilizó la metaetiqueta “Viewport” para hacer que el sitio web sea adaptable a cualquier dispositvo. 

                    <meta name="viewport" content="width=device-width, initial-scale=1.0"> y las mediaqueries con min-width

Para navegar entre secciones, se hizo un “botón hamburguesa”
